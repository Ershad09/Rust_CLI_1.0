
# String Length Counter CLI

A simple command-line tool that measures different types of string length.


## Example

### Basic Input

```
cargo run -- Hello
```

**Output**
<p>Input: Hello

```
Byte Length: 5
Character Count: 5
Visual Length: 5
```

---

### Emoji Input 

cargo run -- 🇮🇳


**Output**
<p>Input: 🇮🇳


    Byte Length: 8
    Character Count: 2
    Visual Length: 1
